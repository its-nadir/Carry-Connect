# Contributing to CarryConnect

Thank you for your interest in contributing!

## How to Contribute

1.  **Fork the repository** and create your branch from `main`.
2.  **Make your changes**. Ensure your code is clean and working.
3.  **Test your changes** to make sure you haven't broken existing functionality.
4.  **Submit a Pull Request** with a clear description of what you've done.

## Reporting Issues

If you find a bug or have a suggestion, please open an issue on GitHub. Provide as much detail as possible to help us understand and resolve the matter.

## Code of Conduct

Please note that this project is released with a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
